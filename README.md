Description

This project is a Password Strength Checker written in Python.
The program evaluates how strong a password is based on multiple security criteria and provides feedback to the user.

It is designed for beginners and showcases practical Python skills such as string handling, conditionals, and user input.

Features

Checks password length

Verifies use of:

Uppercase letters

Lowercase letters

Numbers

Special characters

Rates password strength (Weak / Medium / Strong)

Simple and easy-to-understand logic

Technologies Used

Python 3

How to Run

Make sure Python 3 is installed

Clone the repository:

git clone https://github.com/yourusername/password-strength-checker.git


Navigate to the project folder:

cd password-strength-checker


Run the program:

python password_checker.py

Example Output
Enter your password: P@ssw0rd123
Password strength: Strong
